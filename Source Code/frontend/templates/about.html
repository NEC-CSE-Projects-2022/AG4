About the Project

The Heart Disease Detection System is a web-based deep learning application designed to predict the likelihood of heart disease using essential clinical and physiological parameters.

The predictive model was trained using the Cleveland Heart Disease dataset, one of the most widely used datasets in cardiovascular research. With proper data preprocessing, normalization, and attention-enhanced layers, the system achieves a high level of prediction accuracy.

Dataset Link: Cleveland Heart Disease Dataset (UCI Repository)

This model integrates machine learning and deep learning approaches, leveraging the Bidirectional Long Short-Term Memory and Hybrid Refined Ensemble Framework (BiLSTM + HREF) architecture. The system analyzes features such as age, cholesterol levels, resting blood pressure, ECG readings, and more to identify early signs of heart disease.

The predictive model was trained using the Cleveland Heart Disease dataset, one of the most widely used datasets in cardiovascular research. With proper data preprocessing, normalization, and attention-enhanced layers, the system achieves a high level of prediction accuracy.

Key Features:

  • ✅ Deep Learning–based predictive model (BiLSTM + HREF)
  • ✅ Real-time prediction using user-entered clinical data
  • ✅ Validation for input integrity (no negative values)
  • ✅ Clean, responsive, and user-friendly interface

Our mission is to empower early detection and prevention of heart disease through the use of AI-driven healthcare analytics.
